# road-safety---helmet-detection

The model detects:

Riders
Helmets (Wearing / Not Wearing / Improper)
Number Plates
⚙️ Tech Stack:
Python
YOLOv8 (Ultralytics)
Google Colab (T4 GPU)
OpenCV
📊 Process:
Used a Kaggle dataset for training
Performed data validation and label verification
Trained a YOLOv8 nano model on custom classes
Evaluated using mAP metrics
Tested on unseen images with bounding box detection
🚀 Outcome:

The model successfully detects multiple road objects with good accuracy and demonstrates how AI can be used for road safety applications.
